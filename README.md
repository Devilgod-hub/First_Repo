# Current-Date
## Small app for CI/CD pipeline in Jenkins

## CI/CD Pipeline

This project uses Jenkins automation with GitHub webhooks.

### Workflow
1. Developer pushes code to GitHub
2. GitHub webhook triggers Jenkins automatically
3. Jenkins pulls latest source code
4. Project builds and executes automatically

### Technologies Used
- Jenkins
- GitHub Webhooks
- ngrok (for local webhook exposure)
- Git
- Java
